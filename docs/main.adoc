= Beginning Rust
:author: codcod
:doctype: book

How to compile this book.

[source,bash]
----
$ brew install asciidoctor
$ sudo gem install asciidoctor-epub3 asciidoctor-pdf
$ asciidoctor \
      -r asciidoctor-pdf \
      -b pdf \
      -D build \
      docs/main.adoc
$ asciidoctor-epub3 -D build docs/main.adoc
----
